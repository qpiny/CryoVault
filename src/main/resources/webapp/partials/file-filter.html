Filters

ext(extension)
Select files which ends with extension text
older(duration)
Select files not modified during last duration
newer(duration)
Select files modified during last duration
all
Select all files
none
Select no file

Composition

and(filter1,filter2)
Select files which satisfy filter1 and filter2
or(filter1,filter2)
Select files which satisfy filter1 or filter2
not(filter)
Select files which don't satisfy filter
